This package implements the SuperC parsing engine for parsing all of C.

SuperC Technical Manual

Please read the SuperC technical manual packaged with the source code. 
cd java/src/xtc/lang/cpp

# Depends on pdflatex
make manual.pdf
The manual's source is a latex file called manual.tex.

Quick Start

First setup the environment variables: 
# For bash
. java/src/xtc/lang/cpp/scripts/env.sh
Then run SuperC: 
java xtc.lang.cpp.SuperC program.c
Use -printAST to print the AST. Use -E to just do configuration-preserving preprocessing.
